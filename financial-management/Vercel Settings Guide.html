⚙️ วิธีตั้ง Vercel Settings

ทำให้เว็บแสดงไฟล์ Financial Management.html ได้ถูกต้อง

1

เข้าไปที่ Vercel Dashboard

ไปที่ vercel.com และ log in

ที่หน้า Dashboard คุณจะเห็นรายชื่อ projects ที่สร้างไว้

2

คลิก Project "class-fund-67"

ค้นหาและคลิกที่ project ชื่อ "class-fund-67" (หรือชื่ออื่นที่คุณตั้งไว้)

3

ไปที่ Settings

เมื่อเข้า project แล้ว คุณจะเห็นแท็บด้านบน:

  • Overview (ตำแหน่งปัจจุบัน)
  • Deployments
  • Settings ← คลิกที่นี่

💡 ถ้าไม่เห็น Settings ลองเลื่อนไปขวา หรือเปิดด้านบน

4

เลื่อนลงหา "Build & Development Settings"

ใน Settings page คุณจะเห็นหลายๆ section อยู่เลื่อนลงประมาณ 1/3 จะเห็น section นี้

ดูหลักสังเกต:

  • 🔧 ไอคอน "Build & Development Settings"
  • จะมีปุ่ม "Edit" ถ้าต้อง Edit
5

หา "Output Directory" field

ใน Build & Development Settings จะมี field หลายช่อง:

  • Framework Preset (อาจไม่ต้องแก้)
  • Build Command (ปล่อยว่างไปได้)
  • Output Directory ← ตรงนี้ต้องแก้
  • Install Command (ปล่อยว่างไปได้)
6

กรอก Output Directory = ./

คลิกใน "Output Directory" field และลบข้อมูลที่มี แล้วพิมพ์:

./

นั่นคือ "dot slash" = root directory

7

หา "Root Directory" (ถ้ามี)

บางคนอาจต้องตั้ง "Root Directory" ด้วย:

  • ถ้ามี field "Root Directory" → ตั้งเป็น ./ เหมือนกัน
  • ถ้าไม่มี ก็ข้ามไปได้
💡 หมายเหตุ: Vercel อาจไม่มี "Output Directory" field สำหรับ static sites — ถ้าไม่เห็น ก็ไม่ต้องตั้งก็ได้
8

เลื่อนลงหา "Node.js Version" (ถ้าต้อง)

ไม่ต้องแก้ — เว็บเราใช้ plain HTML + JSX ที่ browser ตีความเอง

9

ลงไปจนเห็น "Advanced" (ไม่ต้องแก้)

ปล่อยไว้อย่างนี้

10

กดปุ่ม Save (ถ้าแก้ไข)

ที่ด้านบนหรือด้านล่างของ Settings page จะมีปุ่ม:

  • Save (ต่อจากแก้ไข)
  • หรือ Deploy (ถ้ากดแล้วจะ redeploy)

กดเพื่อบันทึกการเปลี่ยนแปลง

✅ เสร็จแล้ว! เว็บควรแสดง Financial Management.html ได้ถูกต้องแล้ว — ลองเข้า URL ใหม่ดูครับ

❓ ถ้าไม่เห็น Settings

ลองวิธีนี้:

  • กลับไปหน้า Overview
  • ลองไปที่ Project Settings แบบตรง: vercel.com/[username]/[project-name]/settings
  • หรือคลิกปุ่ม ⚙️ (gear icon) ขวามือบน
